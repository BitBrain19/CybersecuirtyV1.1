## Getting Started

To get started with the SecurityAI Platform, we recommend first reading the [Architecture Overview](./architecture/overview.md) to understand the system's design, followed by the [Deployment Guide](./deployment/deployment_guide.md) to set up your environment.

For new users, the [Getting Started Guide](./user_guides/getting_started.md) provides a comprehensive introduction to using the platform.

For administrators, the [Admin Guide](./user_guides/admin_guide.md) offers detailed information on managing and maintaining the platform.

For developers, the [API Reference](./api/api_reference.md) provides comprehensive information about integrating with the platform.
